===========================
        TIGERFORGE
  Easy Pooling PLUS v.1.0
===========================

- SETUP

1) Just install the Unity Package

- HOW TO USE IT

1) Right-click on your project, in Unity Project panel.
2) In the pop-up menu, navigate to: Create > TigerSoft > Easy Pooling Plus.
3) Click the 'New Pooler' menu voice.
4) A new Pooler (ScriptableObject) will be created in your project.
5) Configure the Pooler settings and use it according to the Manual instructions.
